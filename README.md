# YoutubeBot
A Discord bot that plays Youtube videos - for when the Rythm bot breaks.

## Update
Turns out, this breaks just as much as Rythm used to when I first made this, due to Youtube's tendency to update their API without telling anyone, forcing us (and by "us" I mean "the maintainers of pytube") to try to figure out what they changed and update accordingly. This repository is no longer maintained but it might work if you use a recent version of pytube. No promises though.
